MODULE 1 — Accès à la page admin-tables

     - Se connecter puis accéder à /admin/tables
     - Redirection automatique vers /menu

1.3  Accès avec un compte serveur
     Avec waiter@restoqr.ca : password123
     - Se connecter puis accéder à /admin/tables
     - Redirection automatique vers /menu

1.4  Accès avec un compte admin
     Avec admin@restoqr.ca : password123
     - Se connecter puis accéder à /admin/tables
     - La page d'administration des tables s'affiche correctement
     - Le header affiche « Admin », « Déconnexion », « Quitter »
     - Le bouton « Quitter » permet de revenir à la page précédente

MODULE 2 — Affichage de la liste des tables
================================================================================

2.1  Chargement initial
     Avec admin@restoqr.ca : password123
     - Se connecter et accéder à /admin/tables
     - Un spinner de chargement (mat-spinner) s'affiche pendant le chargement
     - Une fois chargé, les tables s'affichent en grille, triées par numéro

2.2  Affichage des cartes
     - Chaque carte affiche : numéro, capacité, statut (occupée/disponible), QR code
     - Statut : chip « Disponible » (vert) ou « Occupée » (rouge)
     - QR code généré dynamiquement, bouton pour télécharger SVG/PNG
     - Bouton « Copier le lien » : copie l'URL de la table
     - Boutons « modifier » (icône edit) et « supprimer » (icône delete)
     - L'URL de la table est affichée sous le QR code

2.3  État vide
     - Si aucune table n'existe, le message « Aucune table trouvée » s'affiche

================================================================================
MODULE 3 — Création d'une table
================================================================================

3.1  Ouverture de la modal de création
     Avec admin@restoqr.ca : password123
     - Clic sur le bouton « + » à côté du titre : La modal s'ouvre
     - Le titre de la modal est « Ajouter une table »
     - Les champs sont vides (numéro, capacité)
     - Bouton « Annuler » et « Ajouter »

3.2  Validation — Numéro requis
     - Laisser le champ « Numéro » vide et cliquer sur « Ajouter »
     - Message d'erreur : « Le numéro est requis »

3.3  Validation — Numéro trop petit
     - Entrer un numéro < 1
     - Message d'erreur : « Le numéro doit être >= 1 »

3.4  Validation — Numéro trop grand
     - Entrer un numéro > 999
     - Message d'erreur : « Le numéro doit être <= 999 »

3.5  Validation — Numéro en doublon
     - Entrer un numéro déjà utilisé
     - Message d'erreur : « Ce numéro est déjà utilisé »

3.6  Validation — Capacité requise
     - Laisser le champ « Capacité » vide et cliquer sur « Ajouter »
     - Message d'erreur : « La capacité est requise »

3.7  Validation — Capacité trop petite
     - Entrer une capacité < 1
     - Message d'erreur : « La capacité doit être >= 1 »

3.8  Validation — Capacité trop grande
     - Entrer une capacité > 20
     - Message d'erreur : « La capacité doit être <= 20 »

3.9  Création réussie
     - Remplir tous les champs avec des valeurs valides (numéro, capacité)
     - Clic sur « Ajouter »
     - La modal se ferme
     - La nouvelle table apparaît dans la liste

3.10 Annulation
     - Remplir des champs puis cliquer sur « Annuler »
     - La modal se ferme, aucune table n'est créée

================================================================================
MODULE 4 — Modification d'une table
================================================================================

4.1  Ouverture de la modal de modification
     Avec admin@restoqr.ca : password123
     - Clic sur l'icône « edit » d'une table existante
     - La modal s'ouvre avec le titre « Modifier la table »
     - Les champs sont pré-remplis (numéro, capacité)
     - Bouton « Annuler » et « Enregistrer »

4.2  Validations
     - Mêmes validations que la création (3.2 à 3.8)

4.3  Modification réussie
     - Modifier le numéro et/ou la capacité d'une table existante
     - Clic sur « Enregistrer »
     - La modal se ferme
     - La table mise à jour s'affiche dans la liste avec les nouvelles valeurs

4.4  Annulation
     - Modifier des champs puis cliquer sur « Annuler »
     - La modal se ferme, la table n'est pas modifiée

================================================================================
MODULE 5 — Suppression d'une table
================================================================================

5.1  Ouverture de la modal de confirmation
     Avec admin@restoqr.ca : password123
     - Clic sur l'icône « delete » d'une table
     - Une modal de confirmation s'ouvre
     - Le message affiche le numéro de la table à supprimer
     - Bouton « Annuler » et « Supprimer »

5.2  Annulation de la suppression
     - Clic sur « Annuler » : La modal se ferme, la table est toujours dans la liste

5.3  Fermeture par le bouton X
     - Clic sur le bouton « X » : La modal se ferme, la table est toujours dans la liste

5.4  Confirmation de la suppression
     - Clic sur « Supprimer »
     - La modal se ferme
     - La table disparaît de la liste

================================================================================
MODULE 6 — QR code et copie de lien
================================================================================

6.1  Affichage du QR code
     - Chaque carte de table affiche un QR code unique
     - Le QR code correspond à l'URL de la table
     - Le QR code est stylisé selon les paramètres (couleur, forme)

6.2  Copie du lien
     - Clic sur le bouton « Copier le lien » : L'URL de la table est copiée dans le presse-papier
     - Un message « Copié ! » s'affiche

6.3  Téléchargement du QR code
     - Clic sur « Télécharger SVG » ou « Télécharger PNG » : Le QR code est téléchargé

MODULE 10 — Création d'un combo
================================================================================

10.1  Ouverture de la modal de création
     Avec admin@restoqr.ca : password123
     - Clic sur le bouton « + » : La modal s'ouvre
     - Le titre de la modal est « Créer un combo »
     - Les champs sont vides (nom, description, prix)
     - Bouton « Annuler » et « Créer »

10.2  Validation — Nom requis
     - Laisser le champ « Nom » vide et cliquer sur « Créer »
     - Message d'erreur : « Le nom est requis »

10.3  Validation — Nom trop long
     - Entrer un nom > 100 caractères
     - Message d'erreur : « Maximum 100 caractères »

10.4  Validation — Nom composé uniquement d'espaces
     - Entrer des espaces seulement dans le nom
     - Message d'erreur : « Le nom ne peut pas être vide »

10.5  Validation — Description trop longue
     - Entrer une description > 255 caractères
     - Message d'erreur : « Maximum 255 caractères »

10.6  Validation — Prix requis
     - Laisser le champ « Prix » vide et cliquer sur « Créer »
     - Message d'erreur : « Le prix est requis »

10.7  Validation — Prix trop petit
     - Entrer un prix <= 0
     - Message d'erreur : « Le prix doit être supérieur à 0 »

10.8  Validation — Prix trop grand
     - Entrer un prix > 9999.99
     - Message d'erreur : « Le prix ne peut pas dépasser 9999.99 $ »

10.9  Validation — Image trop grande
     - Sélectionner une image > 5 Mo
     - Message d'erreur : « L'image doit être inférieure à 5 MB »

10.10 Création réussie
     - Remplir tous les champs avec des valeurs valides
     - Clic sur « Créer »
     - La modal se ferme
     - Le nouveau combo apparaît dans la liste

10.11 Annulation
     - Remplir des champs puis cliquer sur « Annuler »
     - La modal se ferme, aucun combo n'est créé

MODULE 11 — Gestion des articles d'un combo (combo items)
================================================================================

11.1  Sélection d'un combo
     Avec admin@restoqr.ca : password123
     - Cliquer sur un combo dans la liste
     - Le panneau de détail s'affiche à droite
     - Le titre indique le nom du combo sélectionné
     - La liste des articles du combo s'affiche (ou message vide)

11.2  Ajout d'un article au combo
     - Cliquer sur le bouton « Ajouter un article »
     - La modal de sélection d'article s'ouvre
     - Sélectionner un article dans la liste déroulante
     - Définir la quantité (1-10)
     - Cliquer sur « Ajouter »
     - L'article apparaît dans la liste des articles du combo

11.3  Validation — Article requis
     - Ne pas sélectionner d'article et cliquer sur « Ajouter »
     - Message d'erreur : « L'article est requis »

11.4  Validation — Quantité trop petite
     - Entrer une quantité < 1
     - Message d'erreur : « La quantité doit être >= 1 »

11.5  Validation — Quantité trop grande
     - Entrer une quantité > 10
     - Message d'erreur : « La quantité doit être <= 10 »

11.6  Validation — Article en doublon
     - Ajouter un article déjà présent dans le combo
     - Message d'erreur : « Cet article existe déjà dans ce combo »

11.7  Suppression d'un article
     - Cliquer sur l'icône de suppression d'un article
     - L'article disparaît de la liste

================================================================================
MODULE 12 — Combos dans le menu client
================================================================================

12.1  Affichage des combos
     - Accéder au menu sans être connecté
     - Les combos s'affichent dans la catégorie « Combos »
     - Chaque combo affiche : nom, description, prix, image

12.2  Recherche de combos
     - Taper un terme de recherche dans le champ de recherche
     - Les combos sont filtrés par nom (comme les items)

12.3  Tri par prix
     - Sélectionner « Prix croissant » ou « Prix décroissant »
     - Les combos sont triés par prix

12.4  Filtrage par prix
     - Utiliser les filtres prix min/max
     - Les combos sont filtrés par prix

12.5  Modal de détail du combo
     - Cliquer sur un combo
     - La modal de détail s'ouvre
     - Les articles inclus dans le combo s'affichent
