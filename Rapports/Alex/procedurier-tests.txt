MODULE 2 — Affichage de la liste des tables
================================================================================

2.1  Chargement initial
     Avec admin@restoqr.ca : password123
     - Se connecter et accéder à /admin/tables
     - Un spinner de chargement (mat-spinner) s'affiche pendant le chargement
     - Une fois chargé, les tables s'affichent en grille, triées par numéro

2.2  Affichage des cartes
     - Chaque carte affiche : numéro, capacité, statut (occupée/disponible), QR code
     - Statut : chip « Disponible » (vert) ou « Occupée » (rouge)
     - QR code généré dynamiquement, bouton pour télécharger SVG/PNG
     - Bouton « Copier le lien » : copie l'URL de la table
     - Boutons « modifier » (icône edit) et « supprimer » (icône delete)
     - L'URL de la table est affichée sous le QR code

2.3  État vide
     - Si aucune table n'existe, le message « Aucune table trouvée » s'affiche

================================================================================
MODULE 3 — Création d'une table
================================================================================

3.1  Ouverture de la modal de création
     Avec admin@restoqr.ca : password123
     - Clic sur le bouton « + » à côté du titre : La modal s'ouvre
     - Le titre de la modal est « Ajouter une table »
     - Les champs sont vides (numéro, capacité)
     - Bouton « Annuler » et « Ajouter »

3.2  Validation — Numéro requis
     - Laisser le champ « Numéro » vide et cliquer sur « Ajouter »
     - Message d'erreur : « Le numéro est requis »

3.3  Validation — Numéro trop petit
     - Entrer un numéro < 1
     - Message d'erreur : « Le numéro doit être >= 1 »

3.4  Validation — Numéro trop grand
     - Entrer un numéro > 999
     - Message d'erreur : « Le numéro doit être <= 999 »

3.5  Validation — Numéro en doublon
     - Entrer un numéro déjà utilisé
     - Message d'erreur : « Ce numéro est déjà utilisé »

3.6  Validation — Capacité requise
     - Laisser le champ « Capacité » vide et cliquer sur « Ajouter »
     - Message d'erreur : « La capacité est requise »

3.7  Validation — Capacité trop petite
     - Entrer une capacité < 1
     - Message d'erreur : « La capacité doit être >= 1 »

3.8  Validation — Capacité trop grande
     - Entrer une capacité > 20
     - Message d'erreur : « La capacité doit être <= 20 »

3.9  Création réussie
     - Remplir tous les champs avec des valeurs valides (numéro, capacité)
     - Clic sur « Ajouter »
     - La modal se ferme
     - La nouvelle table apparaît dans la liste

3.10 Annulation
     - Remplir des champs puis cliquer sur « Annuler »
     - La modal se ferme, aucune table n'est créée

================================================================================
MODULE 4 — Modification d'une table
================================================================================

4.1  Ouverture de la modal de modification
     Avec admin@restoqr.ca : password123
     - Clic sur l'icône « edit » d'une table existante
     - La modal s'ouvre avec le titre « Modifier la table »
     - Les champs sont pré-remplis (numéro, capacité)
     - Bouton « Annuler » et « Enregistrer »

4.2  Validations
     - Mêmes validations que la création (3.2 à 3.8)

4.3  Modification réussie
     - Modifier le numéro et/ou la capacité d'une table existante
     - Clic sur « Enregistrer »
     - La modal se ferme
     - La table mise à jour s'affiche dans la liste avec les nouvelles valeurs

4.4  Annulation
     - Modifier des champs puis cliquer sur « Annuler »
     - La modal se ferme, la table n'est pas modifiée

================================================================================
MODULE 5 — Suppression d'une table
================================================================================

5.1  Ouverture de la modal de confirmation
     Avec admin@restoqr.ca : password123
     - Clic sur l'icône « delete » d'une table
     - Une modal de confirmation s'ouvre
     - Le message affiche le numéro de la table à supprimer
     - Bouton « Annuler » et « Supprimer »

5.2  Annulation de la suppression
     - Clic sur « Annuler » : La modal se ferme, la table est toujours dans la liste

5.3  Fermeture par le bouton X
     - Clic sur le bouton « X » : La modal se ferme, la table est toujours dans la liste

5.4  Confirmation de la suppression
     - Clic sur « Supprimer »
     - La modal se ferme
     - La table disparaît de la liste

================================================================================
MODULE 6 — QR code et copie de lien
================================================================================

6.1  Affichage du QR code
     - Chaque carte de table affiche un QR code unique
     - Le QR code correspond à l'URL de la table
     - Le QR code est stylisé selon les paramètres (couleur, forme)

6.2  Copie du lien
     - Clic sur le bouton « Copier le lien » : L'URL de la table est copiée dans le presse-papier
     - Un message « Copié ! » s'affiche

6.3  Téléchargement du QR code
     - Clic sur « Télécharger SVG » ou « Télécharger PNG » : Le QR code est téléchargé

Création d'un combo
================================================================================

Avec admin@restoqr.ca : password123

- Clic sur le bouton « + » à côté du titre : La modal « Créer un combo » s'ouvre
- Les champs sont vides (nom, description, prix)
- La section de disponibilités est présente dans la modal
- Boutons « Annuler » et « Créer » sont visibles

Validations :

- Clic sur « Créer » ET champ « Nom » vide : Message d'erreur « Le nom est requis »
- Entrer un nom > 100 caractères : Message d'erreur « Maximum 100 caractères »
- Entrer uniquement des espaces dans le nom : Message d'erreur « Le nom ne peut pas être vide »
- Entrer une description > 255 caractères : Message d'erreur « Maximum 255 caractères »
- Clic sur « Créer » ET champ « Prix » vide : Message d'erreur « Le prix est requis »
- Entrer un prix <= 0 : Message d'erreur « Le prix doit être supérieur à 0 »
- Entrer un prix > 9999.99 : Message d'erreur « Le prix ne peut pas dépasser 9999.99 $ »
- Sélectionner une image > 5 Mo : Message d'erreur « L'image doit être inférieure à 5 MB »
- Sélectionner un fichier non JPEG/PNG (ex: .gif, .pdf) : Le fichier n'est pas accepté (accept="image/jpeg,image/png")

Création réussie :

- Remplir tous les champs valides ET clic sur « Créer » : La modal se ferme, le combo apparaît dans la liste
- Le bouton affiche « Création... » pendant l'enregistrement
- Sélectionner une image JPEG ou PNG valide : L'aperçu de l'image s'affiche dans la modal

Disponibilités à la création :

- Ajouter une ou plusieurs plages de disponibilité dans la modal : Les disponibilités sont enregistrées avec le combo
- Ne pas ajouter de disponibilité : Le combo est créé mais apparaît avec le badge « Indisponible »

Annulation :

- Remplir des champs ET clic sur « Annuler » : La modal se ferme, aucun combo n'est créé
- Clic sur le bouton « X » de la modal : La modal se ferme, aucun combo n'est créé

================================================================================

Avec admin@restoqr.ca : password123

- Clic sur un combo dans la liste : Le panneau de détail s'affiche
- Le titre indique le nom du combo sélectionné
- Un bouton retour (flèche) permet de revenir à la liste
- Le sous-titre affiche « Gérer les articles de ce combo. »
- Si aucun article : Le message « Aucun article dans ce combo. » s'affiche

================================================================================
Panneau de comparaison de prix
================================================================================

Avec admin@restoqr.ca : password123

- Sélectionner un combo qui contient des articles : Le panneau de comparaison de prix s'affiche
- « Valeur des articles » : La somme des prix individuels des articles × leur quantité
- « Prix du combo » : Le prix du combo
- « Économie » : La différence entre la valeur des articles et le prix du combo (si positive)
- Si la valeur des articles est inférieure ou égale au prix du combo : La ligne « Économie » ne s'affiche pas

================================================================================
Ajout d'un article au combo
================================================================================

Avec admin@restoqr.ca : password123

- Clic sur le bouton « + » dans la vue détail : La modal « Ajouter un article au combo » s'ouvre
- La liste déroulante affiche les articles disponibles
- Un champ de recherche dans la liste déroulante : Les articles sont filtrés en temps réel
- Rechercher un article inexistant : Le message « Aucun article trouvé » s'affiche
- Les articles indisponibles sont grisés et non sélectionnables dans la liste

Validations :

- Clic sur « Ajouter » ET aucun article sélectionné : Message d'erreur « L'item est requis »
- Clic sur « Ajouter » ET quantité vide : Message d'erreur « La quantité est requise »
- Entrer une quantité < 1 : Message d'erreur « La quantité doit être au moins 1 »
- Entrer une quantité > 10 : Message d'erreur « La quantité ne peut pas dépasser 10 »
- Sélectionner un article déjà présent dans le combo : Message d'erreur du serveur

Ajout réussi :

- Sélectionner un article ET quantité valide (1-10) ET clic sur « Ajouter » : L'article apparaît dans la liste
- Chaque article affiché montre : image (ou icône par défaut), nom, quantité
- Le bouton affiche « Ajout... » pendant l'enregistrement

Annulation :

- Remplir des champs ET clic sur « Annuler » : La modal se ferme, aucun article n'est ajouté
- Clic sur le bouton « X » de la modal : La modal se ferme, aucun article n'est ajouté

Combos dans le menu client
================================================================================

- Accéder au menu : Les combos s'affichent dans la catégorie « Combos »
- Chaque carte de combo affiche : nom, description, prix du combo, prix normal barré (somme des articles)
- L'image du combo s'affiche (ou icône par défaut si pas d'image)
- Seuls les combos avec une disponibilité active sont visibles

Recherche et filtres :

- Taper un terme dans le champ de recherche : Les combos sont filtrés par nom
- Sélectionner « Prix croissant » : Les combos sont triés par prix croissant
- Sélectionner « Prix décroissant » : Les combos sont triés par prix décroissant
- Définir un prix min et/ou max : Les combos sont filtrés par prix

================================================================================
Modal de détail du combo (menu client)
================================================================================

Avec client@restoqr.ca : password123

- Clic sur un combo dans le menu : La modal de détail s'ouvre
- La modal affiche : image (ou icône), nom, description
- Section « Ce combo inclut : » : Liste des articles avec quantité (ex: « 2x Burger »)
- Le prix du combo est affiché
- Le prix normal total (somme des articles) est affiché
- Un champ « Instructions spéciales » est disponible (max 255 caractères)
- Un compteur de caractères s'affiche (ex : « 12/255 »)
- Sélecteur de quantité avec boutons +/− : Quantité entre 1 et 50
- Bouton « − » désactivé quand quantité = 1
- Bouton « + » désactivé quand quantité = 50
- Le bouton « Ajouter au panier » affiche le total (prix combo × quantité)
- Clic sur « Ajouter au panier » : Le combo est ajouté au panier avec la bonne quantité et les instructions
- Fermer la modal : Aucun ajout au panier
